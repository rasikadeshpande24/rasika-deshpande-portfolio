# rasika-deshpande-portfolio

This is a simple static website built using Next.js, showcasing the portfolio of Rasika Deshpande.

## Project Structure

```
rasika-deshpande-portfolio
├── app
│   ├── layout.tsx        # Defines the layout component for the application
│   ├── page.tsx          # Main page of the application
│   └── globals.css       # Global CSS styles
├── public
│   └── favicon.ico       # Favicon for the website
├── package.json          # Configuration file for npm
├── next.config.js        # Configuration settings for Next.js
├── tsconfig.json         # TypeScript configuration file
└── README.md             # Documentation for the project
```

## Getting Started

To get started with this project, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd rasika-deshpande-portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

   Open your browser and navigate to `http://localhost:3000` to see your application in action.

To deploy your Next.js application, you can use platforms like Vercel or Netlify. Follow their documentation for deployment instructions.
